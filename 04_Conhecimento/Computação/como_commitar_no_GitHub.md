> Depois de fazer o `git commit -m "Mensagem do commit"`
---
### Faça:
    - `git branch -M main`
    - `git remote add origin https://github.com/usuario/repositorio.git`
---
> Para verificar se deu certo, faça:
> git remote -v
> Deve aparecer a branch remota no github
---
### Para mandar os arquivos para o GitHub
    - `git push -u origin main`
